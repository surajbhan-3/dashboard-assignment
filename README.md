# dashboard-assignment

### Tech stack 

- <b> Frontend <b>
  - Reactjs
  - Formik & Yup for form validation
  - Tabler.io for design
  

- <b> Backend <b>
  - NodeJs
  - Express
  - JWT for authentication
  - Mysql Database 
  - Prisma ORM

- <b> Deployment <b>
  - Vercel for frontend application deployment
  - Aiven for mysql database 
  - AWS (elasticbeanstalk) for node js server
  - openssl for ssl
